## Installation

* `git clone https://github.com/dyyllaann/simplefolio-react.git` this repository
* `cd simplefolio-react`
* `npm install`

## Running / Development

* `npm run dev`
* Visit your app at [http://localhost:3000](http://localhost:3000).

### Linting

* `npm run lint:hbs`
* `npm run lint:js`
* `npm run lint:js -- --fix`

### Building

*  `npm run build`
